🏨 StaySuite - Sistema de gestión de hoteles

StaySuite es una aplicación web desarrollada con Angular (frontend) y Spring Boot (backend), que permite gestionar hoteles, habitaciones y usuarios con autenticación, roles y permisos.

🚀 Funcionalidades principales
🔐 Sistema de autenticación (login y registro)
👥 Gestión de usuarios con roles y permisos
🏨 Creación y administración de hoteles
🛏️ Gestión de habitaciones
🧭 Protección de rutas según roles (guards)
🧱 Arquitectura basada en módulos y componentes reutilizables
🧰 Tecnologías utilizadas
Frontend
Angular
TypeScript
HTML / CSS
Angular Router (guards y rutas protegidas)
Backend
Spring Boot
Spring Security
JPA / Hibernate
MySQL (o la base de datos que uses)


🔐 Seguridad

El sistema incluye:

Autenticación con login
Registro de usuarios
Control de acceso basado en roles
Protección de rutas en frontend y backend
